Declaration:
map<string,int> m; 

Size:
int length=m.size(); 

Insert:
m.insert(make_pair("hello",9));

Erasing an element:
m.erase(val);

Finding an element:

map<string,int>::iterator itr=m.find(val); 

Ex: map<string,int>::iterator itr=m.find("Maps"); //If Maps is not present as the key value then itr==m.end().

Accessing the value stored in the key:
 m["MAPS"]
